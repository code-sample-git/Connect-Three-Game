JavaScript Cheat Sheet for URL Shortening Service

Object: translations

Purpose: Stores language translations for UI elements.

Usage: translations['en-US'].websiteTitle retrieves the website title in English.

Variables

lang: Current language setting.

shortenedUrls: Array to store shortened URL data.

Handling Language Switch

Event Listener: Attached to elements with class .langLink.

Function: updateTranslation(newLang) updates UI elements with selected language translations.

Function: updateTranslation(newLang)

Updates language settings in localStorage. Changes URL to include new language tag. Updates text content of various elements like placeholders, button labels, etc.

Function: handleSubmit(event)

Trigger: On form submission. Validates URL input and sends a POST request to shorten the URL. Handles server response and error situations.

Function: showShortenedURL(data)

Displays the shortened URL and associated options (copy, share, QR code generation). Uses dynamic DOM manipulation to create elements.

Utility Functions

validateUrl(url): Validates the input URL format.

showAlertMessage(message): Displays alert messages.

clearMessages(): Clears alert messages.

Event Listeners

DOMContentLoaded: Sets up initial language settings based on URL, localStorage, or browser settings.

window.onload: Loads shortened URLs from localStorage and displays them.

Key Concepts Demonstrated

Local Storage, Event Handling, Dynamic DOM Manipulation, Asynchronous JavaScript, Error Handling, Localization and Internationalization.

Note

This cheat sheet summarizes the script provided and does not include details of HTML/CSS components that might be required for full functionality.
